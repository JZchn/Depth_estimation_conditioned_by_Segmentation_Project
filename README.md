# Segmentation_conditioned_by_depth_estimation_Medical_Robotics_Project

## Overview
This repository contains a course project implementation for medical image depth estimation conditioned by segmentation

## Pre-trained Models
This project makes use of the following pre-trained models:

- Segment Anything Model (SAM), provided by Meta AI  
- Depth Anything V2 metric, provided by the authors of the Depth Anything project

The pre-trained weights are used as provided by the original authors.
